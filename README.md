# HousieTicketGenerator
Generate Housie Ticket with a custom background image and count of tickets


# MAIN FILES
`generator.py` makes HTMLs in output folder
`html_to_img.py` make IMAGES from HTML in ticket_images folder
`img_to_pdf.py` makes a pdf
`template.html`

--- 

## To change 
- html styles in `template.html`
- Adjust the cell bg, padding, height/width and border

- images bg in `generator.py` as path
- set number of html to be produced in `generator`

- to open output html locally image need to be in output folder
